Backend Services:

* Firebase Authentication
* Firestore Database
* Firebase Storage

---

1. FIREBASE PROJECT SETUP

Step 1:
Go to Firebase Console

https://console.firebase.google.com

Step 2:
Create Project

Project Name:
DevConnect

Step 3:
Register Web App

Copy Firebase Config.

---

2. FIREBASE INSTALLATION

Install Firebase in project

npm install firebase

Create folder:

src/firebase/

Create file:

firebaseConfig.js

Example structure:

import { initializeApp } from "firebase/app";

const firebaseConfig = {
apiKey: "",
authDomain: "",
projectId: "",
storageBucket: "",
messagingSenderId: "",
appId: ""
};

export const app = initializeApp(firebaseConfig);

11. SECURITY RULES (IMPORTANT)

Firestore Rules Concept:

• Anyone can read posts
• Only logged-in users can create posts
• Users can edit only their own posts
• Users can edit their own profile

Example rule logic:

request.auth != null

Meaning:
User must be logged in.

---

12. STORAGE SECURITY RULES

Allow upload only for authenticated users.

Example concept:

allow write: if request.auth != null

Optional:

Limit file size
Limit file type

---
